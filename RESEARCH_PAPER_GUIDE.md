## 🔄 Conversion Guide

### Converting to PDF (Recommended)

#### Method 1: Print to PDF (Windows)

1. **Open RESEARCH_PAPER.html in any web browser**
   - Right-click the file → Open with → Your preferred browser (Chrome, Edge, Firefox)

2. **Print to PDF**
   - Press `Ctrl + P`
   - Select printer: "Print to PDF" or "Save as PDF"
   - Click "Save"
   - Choose your desired location and filename

3. **Verify the PDF**
   - All pages numbered correctly
   - Formatting preserved
   - All sections visible
   - References properly formatted

#### Method 2: Google Chrome Built-in

1. Open HTML file in Chrome
2. Press `Ctrl + P`
3. In the print dialog:
   - Select "Destination": "Save as PDF"
   - Margins: "Default"
   - Scale: 100%
   - Click "Save"

---

### Converting to Microsoft Word

#### Method 1: Copy-Paste (Simple)

1. **Open RESEARCH_PAPER.html in your browser**
2. **Select all content** (`Ctrl + A`)
3. **Copy** (`Ctrl + C`)
4. **Open Microsoft Word** (Create new document)
5. **Paste** (`Ctrl + V`)
6. **Format manually:**
   - Select all text → Font: Times New Roman, Size: 12pt
   - Paragraph spacing: Line spacing 1.5
   - Apply heading styles (Heading 1 for main sections, Heading 2 for subsections)

#### Method 2: Using LibreOffice/OpenOffice (Free Alternative)

1. Open RESEARCH_PAPER.html in LibreOffice Writer
2. File → Save As → Select format: Microsoft Word (.docx)
3. Click Save

#### Method 3: Online Converters

- Use online HTML to DOCX converter
- Upload RESEARCH_PAPER.html
- Download converted Word file
- Verify formatting
